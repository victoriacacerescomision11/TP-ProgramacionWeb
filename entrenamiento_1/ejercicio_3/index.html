Pizza

Para hacer una pizza casera, primero se mezclan harina, agua, levadura y sal hasta formar una masa, luego se amasa y se deja reposar para que leve. Después se estira la masa, se coloca salsa de tomate y queso por encima, y finalmente se lleva al horno hasta que esté bien dorada y lista para servir.

Hamburguesa

Para preparar una hamburguesa, se forma la carne con las manos dándole forma redonda, luego se cocina en una sartén o parrilla hasta que esté bien hecha. Mientras tanto se tuesta el pan, se coloca la carne dentro, se le agrega queso, lechuga y tomate, y se cierra para servir

Indice

Ensalada

Para hacer una ensalada, primero se lavan bien las verduras, luego se cortan en trozos como lechuga, tomate u otros ingredientes. Después se colocan en un bowl, se agrega sal y aceite a gusto, se mezcla todo bien y ya está lista para servir.

Indice

Fideos con tuco

Para preparar fideos con tuco, primero se pone a hervir agua con sal en una olla y, cuando alcanza el punto de ebullición, se agregan los fideos y se cocinan hasta que estén al dente. Mientras tanto, en otra sartén se sofríe cebolla y ajo picados con un poco de aceite, luego se agrega salsa de tomate y se condimenta con sal, pimienta y orégano, dejando cocinar unos minutos para que tome sabor. Una vez listos los fideos, se cuelan y se mezclan con la salsa o se sirven en el plato con el tuco por encima. Se puede agregar queso rallado antes de servir.
