desarrollo web

universidad nacional de la matanza

03/04/2020 - 06/03/2025

Las competencias adquiridas en la carrera de Desarrollo Web en la Universidad Nacional de La Matanza (UNLaM) incluyen el manejo de HTML, CSS y JavaScript para la creación de sitios web, el diseño de interfaces responsivas adaptadas a distintos dispositivos, el uso de buenas prácticas de programación, la estructuración semántica del contenido y la capacidad de desarrollar proyectos web funcionales desde cero. También se desarrollan habilidades para trabajar con herramientas de control de versiones como Git y para resolver problemas básicos de lógica en el desarrollo de aplicaciones web.
